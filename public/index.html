A quick and clean rack project template for Heroku

This is a clean rack project template that I (@javve) use when I create small Heroku projects.

Fork at Github or see it live at http://rack-project-template.herokuapp.com/

It includes some basic folder structure and code for Facebook/Twitter/Gplus/Github sharing.
